# ORF Radiothek / ORF Sound Provider ![Preview image](../assets/icons/orf_radiothek-icon.svg){ width=70 align=right }

Music Assistant support for **ORF Radiothek**, providing access to Austrian radio stations and ORF podcasts.

## Features

|                          |                    |
|--------------------------|--------------------|
| Subscription Required    | No (some content may be region-locked)                 |
| Self-Hosted Local Media  | No                 |
| Media Types Supported    | Radio, Podcasts    |
| Search Supported         | Yes                |
| Radio Mode               | Yes                |

## Content

- **Radio**
  - ORF radio stations
  - Additional private stations included in the ORF Radiothek lineup

- **Podcasts**
  - Official ORF podcasts
  - Includes recently aired radio shows where available

## Configuration

- Optional stream protocol and quality selection for ORF stations
- Optional station filtering

## Notes

- Station logos are shown where available
- Metadata is populated via api were possible
- All content is read-only and streamed directly
