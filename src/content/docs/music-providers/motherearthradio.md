# Mother Earth Radio <img src="/assets/icons/motherearthradio-icon.png" alt="Preview image" style="width: 70px; float: right;"  loading="lazy" />

Music Assistant has support for <a href="https://motherearthradio.de/" target="_blank" rel="noopener noreferrer">Mother Earth Radio</a> — high-resolution internet radio streaming lossless FLAC at up to 192 kHz/24-bit. Contributed and maintained by <a href="https://github.com/fvonhoven" target="_blank" rel="noopener noreferrer">Florian Reiterer</a>

Mother Earth Radio is a one-person, ad-free radio service built around the idea of conscious listening. Every track is hand-picked from vinyl, hi-res downloads, and CD — curated by a professional audio engineer with over 30 years of experience. The music spans an eclectic range from jazz and classical to electronic, blues, and indie — no algorithms, no major-label playlists, just music worth listening to.

> [!TIP]
> **Donate**

## Features

|           |                     |
|:-----------------------|:---------------------:|
| Subscription FREE | Yes |
| Self-Hosted Local Media | No |
| Media Types Supported | Radio |
| [Recommendations](/ui/#view-home) Supported | No |
| Lyrics Supported | No |
| [Radio Mode](/ui/#track-menu) | No |
| Maximum Stream Quality | Lossless FLAC (24 bit, 192 kHz) |
| Login Method | None |

### Channels

- **Mother Earth Radio** — Eclectic audiophile mix from vinyl, hi-res downloads, and CD
- **Mother Earth Instrumental** — Instrumental selections: jazz, electronic, acoustic, ambient
- **Mother Earth Klassik** — Classical music from vinyl and hi-res sources
- **Mother Earth Jazz** — Jazz, funk, and soul

### Other

- When available, details of upcoming tracks will be shown
- Album metadata is enriched with source format, year, and label information (e.g. `[Vinyl · 1998 · !K7]`)

## Configuration

- In the configuration, you only need to click the save button

## Usage

Use the browse view to find stations to add to the library.
